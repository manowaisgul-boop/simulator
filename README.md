A *First-Come, First-Served (FCFS) Scheduling Simulator* built using *HTML, CSS, and JavaScript*.  
This project allows users to add processes with arrival time, burst time, and initial priority, run the FCFS scheduling algorithm, and visualize the execution timeline with animations.

---

## Features

- Add processes with:
  - *Process ID (PID)*
  - *Arrival Time (AT)*
  - *Burst Time (BT)*
  - *Initial Priority* (1 = Highest, 10 = Lowest)
- Run *FCFS Scheduling Algorithm*
- Animated *Execution Timeline*
- View results in a *detailed table* including:
  - Start Time (ST)
  - Completion Time (CT)
  - Waiting Time (WT)
  - Turnaround Time (TAT)
  - Response Time (RT)
- *Reset* button to clear all processes and results
- Toast notifications for successful actions or errors
- Modern *dark-themed UI* with responsive design
